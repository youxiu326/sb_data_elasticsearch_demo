安装Elasticsearch-SQL Site chrome扩展，使用更方便的查询方式：
1).下载扩展ZIP包：https://github.com/shi-yuan/elasticsearch-sql-site-chrome
2).解压。
3).在chrome浏览器输入：chrome://extensions/
4).开启开发者模式。
5).加载已解压的扩展程序。
6).使用时点击扩展图标即可，可以将 SQL 语句翻译成 DSL
